电商项目，技术栈

 > Spring Boot +Dubbo+Zookeeper+Redis+Mybatis+RabbitMQ+MySQL+OSS

## 功能
- 用户注册
- 用户登录
- 首页商品展示
- 商品详情
- 加入购物车
- 购物车商品列表
- 立即购买
- 支付
- 提醒商家发货
- 订单列表
- 站内信模板定制（添加、修改、查询、删除）
- 站内信发送（站内信模板定制、站内信发送明细）
- 订单评论
- 用户积分（积分增加、兑换、积分账户注销、积分账户初始化等）
- 市场营销
